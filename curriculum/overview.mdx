## Philosophy

The AI Researcher Zero-to-SOTA curriculum is a **16-week structured pathway** designed to take you from beginner-level machine learning exposure to research-capable AI engineering. This flagship curriculum assumes no prior deep learning expertise and builds everything from first principles.

## Curriculum Structure

### Core Foundation (Weeks 1-12)

The core foundation provides essential knowledge for modern AI research:

- **Mathematical Foundations**: Optimization, linear algebra, probability
- **Transformer Fundamentals**: Attention mechanics, architecture design, positional encodings
- **Training Systems**: Loss functions, regularization, scaling laws
- **Advanced Topics**: Distributed training, evaluation benchmarks, alignment and safety, interpretability

### Specialized Tracks (Weeks 13-15)

After mastering core concepts, explore specialized domains:

- **Computer Vision Track** (Week 13): Vision transformers and image tokenization
- **Reinforcement Learning Track** (Week 14): RL with transformer-based agents
- **Multimodal Track** (Week 15): Cross-modal alignment and multimodal foundations

### Research Capstone (Week 16)

Culminate your learning with experiment design, paper reproduction, and statistical validation using bootstrap confidence intervals.

## Pedagogy Standard

Every mathematical formula in this curriculum follows a rigorous 6-point standard:

<Accordion title="The 6-Point Math Standard">
  1. **Plain-language intuition** - What does this formula actually do?
  2. **Symbol glossary** with dimensional meaning - Define every variable
  3. **Step-by-step derivation** without skipped algebra - Show all work
  4. **Graph or plotted relationship** - Visualize the concept
  5. **Toy numeric walkthrough** - Concrete example with real numbers
  6. **Formula-to-code mapping** - Direct translation to executable Python
</Accordion>

## Weekly Module Structure

Each week follows a consistent learning pattern:

<Steps>
  <Step title="Read Math Walkthrough">
    Study the core formula with full derivation, graphs, and numeric examples
  </Step>
  <Step title="Run Toy Example">
    Execute `toy_example.py` to verify expected metrics and behavior
  </Step>
  <Step title="Solve Coding Challenge">
    Implement the week's focused challenge in 30-60 minutes
  </Step>
  <Step title="Validate Implementation">
    Unskip exercise tests and ensure your code passes all checks
  </Step>
  <Step title="Compare and Reflect">
    Review the reference solution and write a short reflection
  </Step>
</Steps>

## Learning Modes

The curriculum supports multiple learning paths:

<Tabs>
  <Tab title="Core Sequential">
    Follow weeks 1-16 in order for a complete foundation in modern AI research. This is the recommended path for most learners.
    
    **Time Commitment**: 16 weeks at ~10-15 hours per week
  </Tab>
  <Tab title="Realm-Augmented">
    After completing core weeks, dive deep into specialized realm guides that expand on specific topics with project ladders and advanced implementations.
    
    **Best For**: Learners who want to specialize in specific domains like vision, RL, or LLM systems
  </Tab>
  <Tab title="Research-First">
    For experienced practitioners: Skip to weeks 9-16 for advanced topics, then backfill foundational weeks as needed.
    
    **Prerequisites**: Strong Python skills, basic ML exposure, comfort with mathematical notation
  </Tab>
</Tabs>

## Module Artifacts

Every week includes these required files:

| Artifact | Purpose |
|----------|--------|
| `README.md` | Module overview, learning outcomes, and workflow |
| `math_walkthrough.md` | Full derivation following the 6-point standard |
| `diagram.mmd` | Visual architecture or concept diagram |
| `toy_example.py` | Runnable example demonstrating core concepts |
| `coding_problem.md` | Week's focused coding challenge prompt |
| `problem.py` | Starter implementation for learners |
| `solution.py` | Reference implementation |
| `tests/` | Comprehensive test suite for validation |
