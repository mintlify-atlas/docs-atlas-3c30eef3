## Introduction

The `ai-researcher` CLI provides command-line tools for managing curriculum utilities, running experiments, and performing quality checks. The CLI is designed for toy-safe experimentation and educational purposes.

## Installation

The CLI is available after installing the `ai-researcher-repo` package:

```bash
pip install -e .
```

Once installed, you can access the CLI using:

```bash
ai-researcher --help
```

## Quick Start

```bash
# Prepare data for experiments
ai-researcher prepare-data

# Train a toy language model
ai-researcher train-toy

# Evaluate model performance
ai-researcher eval

# Run quality checks
ai-researcher run-checks
```

## Command Structure

All commands follow the pattern:

```bash
ai-researcher <command> [options]
```

## Exit Codes

The CLI uses standard exit codes:
- `0`: Success
- Non-zero: Error (specific code depends on the failure)

## Configuration

Some commands accept a `--config` parameter to specify custom repository configuration. If not provided, default configuration is used.
